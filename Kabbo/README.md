# Kabbo AI

Simple local AI assistant using Ollama + Python.

## Setup

### 1. Create Virtual Environment

Windows:
python -m venv venv

### 2. Activate Environment

PowerShell:
.\venv\Scripts\Activate.ps1

### 3. Install Packages

pip install -r requirements.txt

### 4. Install Ollama

https://ollama.com

### 5. Download Model

ollama run mistral

### 6. Run Kabbo

python backend/kabbo.py
